# ☀️ SolarCalc - Simulador de Viabilidade Fotovoltaica

## Visão do Produto
Para proprietários de residências e pequenos comércios que desejam reduzir custos de energia, o SolarCalc é um sistema de apoio à decisão que avalia a viabilidade da energia fotovoltaica. Diferente de calculadoras genéricas, ele oferece um dashboard personalizado com projeções de longo prazo, considerando variações geográficas e técnicas reais, permitindo que o usuário invista com segurança.
